公司簡介

誠實創新

鈺剛實業有限公司是台灣專業鋼格板製造商，其生產規模及年出口量均列同行並列前茅。
公司始創於1997年，佔地面積21500m²、廠房面積9600m²，本公司擁有先進的設備、成熟的工藝及十多年專業壓焊鋼隔板的生產和管理經驗，來從事扁鋼生產到鍍鋅鋼格板成品生產的企業。

本公司採用全自動高壓電阻熔接機生產，年產鋼格板及其相關產品(樓梯踏板、鋼架平台格柵板、不銹鋼格柵板…等)年產量15000噸。產品銷往國外的有美洲、中東、大洋洲、東南亞…等各國。在國內的大型公共工程有台灣高鐵、台中捷運、台北捷運、台積電、台塑六輕、中鋼、中油…等都能見到鈺剛格柵板的身影。

公司於2013年通過ISO9001，鈺剛實業有限公司一向秉持的是完美品質、精良的製作、交貨的迅速、合理的價格及處處用心的經營理念來為各界服務皆是鈺剛公司所堅持的理念。
